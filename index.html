Layout

You can align HTML elements horizontally by giving their parent element the dm-layout-horizontal class name. Its childs will be aligned horizontally instead of vertically.

The dm-layout-horizontal element is split in 12 columns. You can specify how many columns one of its childs may take up with the dm-width="N" attribute.

1
2
9

This example is what this code outputs:

<div class="dm-layout-horizontal dm-layout-center" dm-text-color="white" dm-spacing="small" dm-font-size="24"> <!-- The .dm-box class name gives the element a minimum width and height and some padding --> <!-- The .dm-layout-center centers the elements within it, so we encapsulate the number in a <span> --> <div class="dm-box dm-layout-center" dm-size="small" dm-width="1" dm-background-color="yellow" dm-border-radius="small"> <span>1</span> </div> <div class="dm-box dm-layout-center" dm-size="small" dm-width="2" dm-background-color="green" dm-border-radius="small"> <span>2</span> </div> <div class="dm-box dm-layout-center" dm-size="small" dm-width="9" dm-background-color="cyan" dm-border-radius="small"> <span>9</span> </div> </div>
